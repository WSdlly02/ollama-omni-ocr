# Ollama Omni-OCR

This contains everything you need to run your app locally using Ollama.

## Run Locally

**Prerequisites:**
1. Node.js
2. [Ollama](https://ollama.com/) installed and running.
3. Pull the model: `ollama pull qwen3-vl:8b`

**Steps:**

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
